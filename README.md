# Bookstore

## About The Project

A backend bookstore application developed in Kotlin and Spring Boot

## Technologies used: 

  * Kotlin
  * Spring Boot
  * JPA Hibernate
  * Rest API
  * Spring Security OAuth2
  * Swagger
  * MySQL
  * Maven
  * Ktlint formatter (maven plugin and dependency)
  * NEXUS Repository Manager
